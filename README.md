# Rust Raid Calculator

Web application for calculating raid costs in the game Rust.

This project was created as a semester assignment for the **Frontend Technologies** course.  
The application is built using **Vue.js 3** with **Options API**, **Pinia** for state management, and **Vue Router**.

---

## Technologies Used

- Vue.js 3  
- Options API  
- Pinia  
- Vue Router  
- TypeScript  
- Vite  
- HTML5, CSS3  

---

## Features

- Multiple pages (Single Page Application)
- Raid cost calculation for different structures
- Reusable Vue components
- Centralized state management with Pinia
- Simple and clear user interface

---

## Project Structure

```
src/
├── components/    # Reusable components
├── views/         # Application pages
├── stores/        # Pinia stores
├── router/        # Vue Router configuration
├── App.vue        # Root component
└── main.ts        # Application entry point
```

---

## Installation & Run

```bash
npm install
npm run dev
```
